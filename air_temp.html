ERA5 Surface Air Temperature Maps. Single and Multiple Options

Generate high-resolution surface air temperature (t2m) maps directly from ERA5 hourly surface data. These tools require ERA5 2m temperature datasets as input and allow you to select the region, time period, and visual style. The output includes clear maps and animations that highlight both spatial temperature patterns and their temporal evolution. ERA5 2m temperature datasets are freely available from the Copernicus Climate Data Store (CDS).

Single Map

Create a single ERA5 2m temperature map for a selected region and time. Simply define the latitude and longitude boundaries, select the date and hour, and choose a colormap for the visualization. The resulting map can be exported as both PNG and PDF.

Multiple Maps

Using these tools, produce a series of ERA5 2m surface temperature (T2M) maps between a specified start and end date and time. Define the latitude and longitude boundaries, time interval, colormap, output filenames, and titles for each map. The maps can be exported as individual PNGs, combined into a single PDF, and automatically compiled into an animated GIF and MP4 video, providing a clear visualization of temporal evolution.
